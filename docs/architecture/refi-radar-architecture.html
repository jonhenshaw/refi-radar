Refi Radar architecture

Cloudflare-first mortgage-rate intelligence dashboard

A map of the actual codebase and production infrastructure: React/Vite on Cloudflare Pages, a Hono Worker API, D1 durable storage, KV snapshot cache, scheduled collectors, public financial/news feeds, and browser-local alert rules.

Runtime flow

  • Browser loads static React assets from refi-radar.pages.dev.
  • Vite build-time VITE_API_BASE points API calls at the Worker.
  • Dashboard calls /api/latest every 60s and /api/series/compare when the chart range changes.
  • If the API is empty/unavailable, the UI deliberately falls back to demo data and shows a sample-data warning.

Ingestion flow

  • Cloudflare cron runs the Worker every 15 minutes: */15 * * * *.
  • collectAllSources(env) fetches MND, FRED, news RSS, and curated calendar sources.
  • Collectors normalize each record into shared TypeScript shapes, then write to D1.
  • After ingestion, the Worker rebuilds latest-snapshot in KV with a 90-second TTL.

Data stores

  • D1 refi-radar-prod holds durable tables: sources, rate observations, news, calendar, loan profiles, alerts.
  • KV REFI_RADAR_CACHE is read-heavy/ephemeral and keeps the homepage fast.
  • Browser localStorage currently owns the MVP alert rules and alert events.
  • @refi-radar/shared is the typed contract between Worker and Web.

Current foot-guns

  • Worker invocations have API/subrequest limits — scheduled collectors must batch writes and avoid historical replays.
  • Some public feeds are fragile: MND scraping/RSS and Treasury feed timeouts can fail independently.
  • D1 schema migrations are manual right now because the CI token lacks D1 edit permission.
Plain-English version: the app is a static dashboard hosted by Cloudflare Pages. It talks to a separate Cloudflare Worker API. The Worker both serves reads and runs the scheduled collector. Durable history lives in D1; the latest homepage payload is cached in KV; alert rules are still browser-local. The recent “no headlines/calendar” issue happened in the ingestion leg: the collector exceeded Worker limits before later sources were written.
