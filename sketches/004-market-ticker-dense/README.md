## Variant: Dense Market Ticker

### Design stance
Keep the winning Market Ticker direction, but compress vertical rhythm so more useful decision/context data appears above the fold.

### Key choices
- Layout: hero rate + move, three micro-stat chips, compact chart, dense source rows.
- Typography: primary number remains large, but reduced from the original to reclaim space.
- Color: same dark fintech palette with blue/green signal accents.
- Interaction: range pills stay in the chart header; source rows remain tappable; source health is demoted to a tiny status row.

### Trade-offs
- Strong at: information density, repeated mobile checking, fewer scrolls.
- Weak at: less airy/premium than the first mockup; small text needs accessibility care.

### Best for
- The production mobile dashboard if we want the first mockup’s direction but a sharper, more compact finance-app feel.
