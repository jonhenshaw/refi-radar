## Variant: Card Carousel

### Design stance
Preserve the existing cinematic cards, but make source selection mobile-native.

### Key choices
- Layout: horizontal metric card deck controls one focused chart panel.
- Typography: big rate numbers retained, but vertical stacking removed.
- Color: dark glass cards with selected-state blue glow.
- Interaction: swipe source cards, chart updates, alerts tied to selected source.

### Trade-offs
- Strong at: keeping current brand feel, touch interaction, source focus.
- Weak at: carousel affordance can hide secondary sources.

### Best for
- A compromise between the current aesthetic and a cleaner mobile flow.
